A savoir:
Projet sur vue 3;
Pour lancer les client et le serveur sur la même vm utiliser "screen":
- screen -S nom_de_la_session (sauvegarder le terminal)
- CTRL A + CTRL D (sortir du terminal)
- screen -r nom_de_la_session (retourner dans le terminal)
